<h2 class="wp-block-heading">Project Overview2</h2>



<p></p>



<p></p>



<p></p>



<p></p>



<p></p>



<p>Japanary&nbsp;is a sophisticated web application I developed to streamline the Japanese language learning process. Born from my personal experience as a Japanese language learner, I&nbsp;identified a common pain point: the time-consuming process of creating Anki flashcards. The&nbsp;application simplifies this workflow by providing an all-in-one solution for dictionary lookup, flashcard creation, and example sentence generation.</p>



<h2 class="wp-block-heading">Technical Stack &amp; Architecture</h2>



<h3 class="wp-block-heading">Core&nbsp;Technologies</h3>



<ul class="wp-block-list">
<li>React 18: Chosen as the primary frontend framework for its robust ecosystem and efficient rendering capabilities</li>



<li>TypeScript: Implemented for type safety and improved developer experience</li>



<li>Vite: Used as the build tool for its exceptional development experience and fast build times</li>
</ul>



<h3 class="wp-block-heading">UI/UX Components</h3>



<ul class="wp-block-list">
<li>TailwindCSS: Utilized for responsive and modern styling with utility-first approach</li>



<li>Shadcn UI: Implemented for accessible and customizable UI components</li>



<li>Radix UI: Integrated for building a robust component system with excellent accessibility</li>
</ul>



<h3 class="wp-block-heading">State&nbsp;Management &amp; Data Fetching</h3>



<ul class="wp-block-list">
<li>TanStack Query (React Query): Employed for efficient server state management and data fetching</li>



<li>React Hook Form: Implemented for form handling with built-in validation</li>



<li>Firebase: Used for backend services and authentication</li>
</ul>



<h3 class="wp-block-heading">Testing &amp; Quality Assurance</h3>



<ul class="wp-block-list">
<li>Vitest: Chosen as the testing framework for its speed and compatibility with Vite</li>



<li>React Testing Library: Implemented for component testing with a user-centric approach</li>



<li>ESLint &amp; TypeScript: Used for code quality and type checking</li>



<li>Husky: Implemented for pre-commit hooks to ensure code quality</li>
</ul>



<h2 class="wp-block-heading">Key Features</h2>



<ol class="wp-block-list">
<li>One-Click Flashcard Creation: Streamlined the process of creating Anki flashcards from Japanese vocabulary</li>



<li>Integrated Dictionary Lookup: Immediate access to word definitions and readings</li>



<li>AI-Powered Enhancements: Integration with OpenAI API for generating example sentences and additional context</li>



<li>Custom Content Management: Ability to add images and customize flashcard content</li>



<li>Modern, Responsive Interface: Built with TailwindCSS for a clean, user-friendly experience</li>
</ol>

<li>Integrating multiple third-party APIs while maintaining performance</li>



<li>Building a responsive and accessible UI that works across devices</li>



<li>Setting up a comprehensive testing suite for reliable deployments</li>
</ul>

</ul>
