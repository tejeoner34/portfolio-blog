Expense Tracker Web Application

By atejezar•23/04/2025

Project Overview

A modern, full-stack expense tracking application built with Next.js, featuring real-time expense management, currency conversion, and comprehensive data visualization. The application helps users track their expenses across different categories and currencies, with automatic conversion to their default currency.

Technical Stack

  • Frontend Framework: Next.js 15.2.1 with React 19
  • Styling: Tailwind CSS with custom animations
  • Database: PostgreSQL with Prisma ORM
  • Authentication: NextAuth.js with credentials provider
  • Data Visualization: Recharts
  • Form Handling: React Hook Form with Zod validation
  • State Management: React Query (TanStack Query)
  • UI Components: Custom components built with Radix UI primitives
  • Testing: Jest with React Testing Library
  • Type Safety: TypeScript
  • Code Quality: ESLint with Husky pre-commit hooks

Key Features

  1. User Authentication
  • Secure sign-up and login system

  1. Expense Management
